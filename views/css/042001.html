    # 半像素细线/边框

    # 方案 1：

    设置 1px 的线，然后给颜色增加透明度，使颜色变浅。

    因为屏幕渲染的最低限度其实就是 1px，而更细的线，只是在靠近边缘的部分颜色浅淡罢了。1px 的浅色线看起来比深色要细，这是利用的视觉效果。

    比如原本是 border: 1px solid rgb(112, 112, 112); 可以改成 border: 1px solid rgba(112, 112, 112, 0.5);

    # 方案 2：

    通过变换缩放，为了方便多数情况下只需要固定缩放 0.5 即可，但有时为了更好的效果，可能要根据不同设备的 DPI，设置多套缩放比。

    DPI 在 4 及以上时，缩放 0.25；在区间 [3, 4) 内时，缩放 0.33；在区间 [2, 3) 内时，缩放 0.5。

    示例如下：

      /* 横向细线 */
  .line--retina {
    width: 100%;
    height: 1px;
    background: #dfdfdf;
    transform: scaleY(0.5);
    /* 根据具体情况，设置缩放基点, 这里设置为左上角不动，从右下角向左上角收缩，因为是 scaleY 只缩放Y轴，所以是从下向上收缩 */
    transform-origin: 0 0;
  }

  /* 四边框细线 */
  .has-retina-border {
    /* ... 内容其他样式，如width/height/font-size等 */
    position: relative;

    &::before {
      position: absolute;
      left: 0;
      top: 0;
      width: 200%;
      height: 200%;
      border: 1px solid #df3348;
      /* 如果有圆角 也要放大 2 倍 比如原本需要的圆角是 8px */
      border-radius: 16px;
      transform: scale(0.5);
      transform-origin: 0 0;
    }
  }

  /* 根据DPI设置的，需要多个选择器分别设置缩放级别，如下情况，标签有一个基础类名 line--base，而后根据不同的 DPI 添加dpi相关的类名 */
  .line--base {
    width: 100%;
    height: 1px;
    background: #dfdfdf;
    transform-origin: 0 0;
  }

  .line--dpi2 {
    transform: scaleY(0.5);
  }

  .line--dpi3 {
    transform: scaleY(0.33);
  }

  .line--dpi4 {
    transform: scaleY(0.25);
  }

    # 小标签文字垂直居中

    这种情况是一个大难题，没有谁能完美解答。除非自带字体包，并设置使用自带的字体，然后根据字体进行代码的精细调节。

    主要原因是不同的字体族，在同一个字号下，显示的大小不尽相同；不同的字体族基线也不同，同一个行高结果也会有差别。

    所以这一点必须要跟设计说清楚，要做到文字垂直居中，只能限定在一种字体下。

    然后我们再以设计常用的苹方字体来讨论。

    由于浏览器有限制，字体的最小显示字号为 12px（当然，小程序/APP 并没有这个限制），如果需要字体更小，那么也需要使用变换来缩放。

    而且在小尺寸下垂直居中不好做，也可以放大尺寸做垂直居中，然后再通过变换缩小，这样在小尺寸下的居中情况看起来会更好一点。

    示例：

      .tag {
  /_ 以下尺寸都是 2 倍尺寸 _/
  height: 32px; /_ 原本尺寸可能是 16px 的高 _/
  padding: 4px 8px;
  box-sizing: border-box;
  border-radius: 8px;
  font-size: 24px;
  line-height: 24px; /_ 32 - 4 - 4 = 24; 如果没有设置 box-sizing: border-box; line-height 应该设置 32px 即与高相同 _/
  background: #df3348;
  color: #fff;
  transform: scale(0.5);
  transform-origin: 0 0;
  }

    # 文字溢出隐藏并在最后显示省略号

    这个要区分一下是单行文本，还是多行文本。

    虽然多行文本的方式也适用于单行的情况，但是单行的情况下，还是建议使用另一种方式，原生支持，兼容性更好。

    代码：

      /_ 单行溢出省略显示 _/
  .single-line {
  /_ 必须要有一个宽度范围，不然无法确定如何算作超出,即使 width: auto; 也行，只要它的父容器有定宽 _/
  /_ 同时这也隐性的说明了一件事，行内元素无法做文字溢出，因为它不能设置宽度 _/
  width: 100px;
  overflow: hidden; /_ 确定文本溢出后需要隐藏 _/
  white-space: nowrap; /_ 强制要求文字不能换行，即单行显示 _/
  text-overflow: ellipsis; /_ 确定文本的隐藏形式，省略号 _/
  }

  /_ 多行溢出省略显示 _/
  .multi-line {
  display: -webkit-box; /_ 定义元素为 -webkit-box，这里从前缀也能知道，它是有兼容性的 _/
  -webkit-box-orient: vertical; /_ 设置方向为垂直方向 _/
  -webkit-line-clamp: 2; /_ 设置行数量，如果是超出 3 行省略，那就设置为 3 _/
  overflow: hidden;
  text-overflow: ellipsis;
  }

    # 两边垂直居中的横线与居中的文字

    除了使用 flex 布局，两边各一个横线，中间添加文字以外，这里提供另一种思路。

    客户所见的并不一定就是前端实际所做的，前端 er 应该把自己想象成一个魔术师，只要给观众看到想让他看到的就行了。

    从外观效果上看是有左右两条横线，但是我们可以只用一条线来做，只要给中间的文字加上与底色相同的背景，覆盖线的中间部分，那么看起来不就是两条线了吗？

      <view class="title">
    <text class="title-text">标题文案</text>
  </view>
  .title {
    position: relative;
    width: 300px;
    height: 60px;
    background: #fff;

    &::before {
      content: '';
      position: absolute;
      left: 0;
      top: 50%;
      transform: translateY(-50%);
      width: 100%;
      height: 1px;
      background: #df3348;
    }

    &-text {
      position: absolute;
      left: 50%;
      top: 50%;
      z-index: 1;
      transform: translate(-50%, -50%);
      // 用 padding 多覆盖一部分，使得文字与线之间有一部分间距
      padding-left: 12px;
      padding-right: 12px;
      background: #fff;
      color: #df3348;
    }
  }

    # 设置等比例宽高的容器

    如果一个容器的高度，需要随着宽度的改变而等比例的改变，除了使用 JS 计算设置以外，还可以仅通过 CSS 样式来实现。

    这里可以利用一个 margin/padding 的特性来实现。

    小知识：在 CSS 中，margin/padding 的值如果是百分比，那么不论是 left/right 还是 top/bottom 都是依照着父容器的 width 来计算的。

    不要理所当然的认为 left/right 是以父容器宽度计算，而 top/bottom 是以父容器的高度计算【这是错误的】。

      .rect {
  width: 80%; /_ 宽度可能是不定的，这里设置是父容器宽度的 80% _/
  height: 0; /_ 设置高度为 0 _/
  padding-top: 40%; /_ 由 padding-top 撑起容器高度，而因为 padding-top 的百分比是父容器宽度计算，所以这就是一个 宽高比 2:1 的容器 _/
  /_ 如果它还有子元素，并且要展示内容怎么办？ _/
  position: relative; /_ 设置定位 _/

    .inner {
    /_ 子元素用 absolute 定位 _/
    position: absolute;
    top: 0;
    left: 0;
    width: 100%;
    height: 100%;
    }
  }
